# Wolf3dX
Open source Monogame implementation of Wolfestein 3d. It does requiere to have game files but can work with Shareware version.

## Prerequisites

   
    -Windows
    MonoGame: 3.4 or higher
    OpenTK 1.1
    
    -Linux
    OpenGL 3.0
    SDL2
